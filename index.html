Paddy-Editor

An open-source project-oriented
2D game editor

Get Started Github

NODES EDITOR

Paddy-Editor have nodes editor that helps create gameplay logic in. It's comes with few basics nodes. Though, it is completely possible to create your own custom node.

INTEGRATION

Paddy-Editor saves to Json file. Which can be easily parsed to be used in any 2d framework/engine

PLUGINS

Paddy-Editor have plugins system that give you full control over the editor. Which can be used to create new custom nodes or UI or custom exporter for your own framework/engine. Plugins are written in javascript
